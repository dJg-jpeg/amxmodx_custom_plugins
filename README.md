# Round End Block Bot (AMXX)

Prevents round ending by creating invisible immortal bots.

## Requirements

- AMX Mod X 1.9+
- Modules:
  - fakemeta
  - hamsandwich
  - cstrike

## Installation

1. Compile .sma using AMXX compiler
2. Copy .amx to addons/amxmodx/plugins
3. Enable required modules in addons/amxmodx/configs/modules.ini
4. Add plugin to plugins.ini
